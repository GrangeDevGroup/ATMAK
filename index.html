ATMAK
Android Tactical Multipurpose Assessment Kit

ATMAK is a tactical situational awareness and mission coordination platform designed for Android devices and cross-platform environments. It combines mapping, operational planning, field intelligence, and team coordination into a single modular system inspired by modern tactical platforms.

Download on GitHub Explore Features

Core System

Interactive Tactical Maps

Advanced mapping layers, GPS tracking, coordinate systems, and navigation tools.

Mission Planning

PACE planning, SITREPs, checkpoints, and operational planning workflows.

Field Intelligence

Intel collection, incident reporting, evidence tracking, and investigative tools.

Team Coordination

Team rosters, live GPS synchronization, tasks, and operational collaboration.

Offline Operation

Fully functional offline-first design with optional synchronization.

Secure Data Handling

Encrypted backups, import/export tools, and peer-to-peer transfers.

Map System

  • 19+ map layers including satellite, terrain, ESRI, USGS
  • Waypoints, markers, shapes and tactical symbols
  • Distance and area measurement tools
  • Track recording and breadcrumb trails
  • Live coordinate formats: MGRS, UTM, Maidenhead, DMS
  • KML and GPX import/export
  • Customizable map toolbar

Team & Coordination

  • Team rosters and member roles
  • Live location sharing
  • Peer-to-peer synchronization
  • Mission checkpoints and progress tracking

Platform Support

  • Android (APK / AAB)
  • Web deployment
  • Windows (EXE / MSI / AppX)
  • Linux (tar.gz / deb)
  • Responsive layouts for phones, tablets, and desktops
